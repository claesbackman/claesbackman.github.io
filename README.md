# claesbackman.github.io

Personal academic website for Claes Bäckman — served by GitHub Pages at <https://claesbackman.github.io/>.

Plain static HTML/CSS. To preview locally:

```
python3 -m http.server 8000
```

Then open <http://localhost:8000/>.

## Files

- `index.html` — single-page site
- `assets/style.css` — styles
- `assets/photo.jpg` — profile photo (add this file)
- `CV_ClaesBackman.pdf` — linked from the site
- `Papers/` — paper PDFs referenced from the site (add as needed)
